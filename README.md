# 🎓 University Management & Experience Platform

A modern, scalable, and conversion-focused **University Web Application** built using the **MERN stack**, designed to serve applicants, parents, visitors, and placement stakeholders.

This platform acts as a **digital ecosystem for a university**, combining academics, admissions, placements, campus life, and more into a seamless experience.

---

## 🚀 Features

### 🏠 Home Dashboard

* Hero section with CTA (Apply Now / Brochure)
* Rankings & Achievements
* Placement highlights
* Facilities overview
* News & Announcements
* Upcoming Events
* Campus life preview
* Testimonials & alumni insights

### 🏛️ About

* University overview
* Mission & Vision
* Leadership profiles
* Accreditations & recognitions

### 🏢 Departments

* Department-wise structure (CSE, ECE, etc.)
* Faculty details
* Labs & facilities
* Courses offered

### 📚 Academics

* UG / PG / PhD programs
* Curriculum & syllabus
* Academic calendar
* Examination details

### 📝 Admissions

* Step-by-step admission process
* Eligibility criteria
* Fee structure
* Scholarships
* Application CTA (conversion-focused UI)

### 💼 Placements

* Placement statistics (highest, average package)
* Recruiter network
* Internship opportunities
* Success stories

### 🔬 Research

* Research domains
* Publications & patents
* Ongoing projects
* Industry collaborations

### 🌿 Campus Life

* Clubs & societies
* Events & fests
* Hostel & sports facilities
* Campus gallery

### 🤝 Alumni

* Notable alumni
* Alumni stories
* Mentorship & networking

### 📞 Contact

* Contact form
* Location map
* Department-wise contacts

## ⚡ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/university-platform.git
cd university-platform
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Open in browser

```
http://localhost:5173
```
